# Robespierre — 10AMPRO Collective Intelligence

WhatsApp group engagement analytics dashboard for the 10AMPRO community.

## Features

- **🧠 Intelligence** — System health gauges, KPIs, learning flywheel
- **💡 Insights** — Founder dependency analysis, bus factor, deep dives
- **⚔️ Robespierre** — The guillotine: zombies, deserters, death row, source quality
- **👥 Members** — Full ranked list with search, filter, expandable scores

## Stack

- Next.js 14 (App Router)
- React 18
- No external UI libs — pure inline styles
- JetBrains Mono + Inter fonts

## Data

126 members analyzed across 96 days (Nov 16 '25 → Feb 21 '26).
39 zombies (zero posts), 9 Tier C, 5 dormant Tier B = 53 to cut.
